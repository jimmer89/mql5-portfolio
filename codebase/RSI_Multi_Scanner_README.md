# RSI Multi-Symbol Scanner

## Description

**Scan multiple currency pairs for RSI trading opportunities in real-time.**

This indicator displays a compact dashboard showing RSI values for up to 28 symbols simultaneously. Instantly spot overbought and oversold conditions across your watchlist without switching charts.

### Features

✅ **Multi-Symbol Scanning** - Monitor up to 28 pairs from a single chart
✅ **Real-Time Updates** - RSI values update on every tick
✅ **Visual Signals** - Color-coded BUY/SELL signals (green for oversold, red for overbought)
✅ **Customizable Alerts** - Popup, sound, and push notifications
✅ **Clean Dashboard** - Minimal, non-intrusive panel design
✅ **Fully Configurable** - Adjust RSI period, levels, colors, and position

### Default Symbols

EURUSD, GBPUSD, USDJPY, AUDUSD, USDCAD, NZDUSD, USDCHF, EURJPY

(Easily customizable to any symbols your broker offers)

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| Symbols | Major pairs | Comma-separated list of symbols to scan |
| RSI Period | 14 | Standard RSI calculation period |
| Overbought | 70 | Level for sell signals |
| Oversold | 30 | Level for buy signals |
| Alert Popup | true | Show popup alerts |
| Alert Sound | true | Play sound on signals |
| Alert Push | false | Send push notifications |

### How to Use

1. Attach to any chart (the scanned symbols are independent)
2. Customize the symbol list to match your watchlist
3. Watch the dashboard for BUY (green) and SELL (red) signals
4. Click on an alert to investigate the opportunity

### Best Practices

- Use as a **screening tool**, not a standalone trading system
- Combine with price action confirmation
- Works best on H1 and H4 timeframes for swing trading
- For scalping, consider lower RSI period (7-10)
