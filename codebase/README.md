# MQL5 CodeBase Submissions

Free indicators and tools for MQL5 CodeBase publication.

## Available

### RSI Multi-Symbol Scanner v1.10

Monitor RSI values across multiple currency pairs in real-time from a single chart.

**Features:**
- Scans up to 28 symbols simultaneously
- Selectable timeframe (independent of chart)
- Color-coded BUY/SELL signals
- Customizable alerts (popup, sound, push)
- Clean, compact dashboard design
- Alert spam prevention (one per symbol per bar)

**Files:**
- `RSI_Multi_Scanner.mq5` — Main indicator code
- `RSI_Multi_Scanner_README.md` — Documentation

**Recommended usage:** Any chart, H1 or H4 timeframe for swing trading signals.

---

## Code Guidelines Applied

All code follows [MQL5 CodeBase guidelines](https://www.mql5.com/en/articles/19441):
- Formatted with MetaEditor Styler
- 3-line comment blocks for functions
- `//---` comments for key stages
- English comments throughout
- Latin characters in filenames
- Input parameters documented
